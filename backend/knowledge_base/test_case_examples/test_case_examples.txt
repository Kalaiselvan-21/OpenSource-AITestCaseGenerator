**Scenario 1: New User on Select Form Type Page (With or Without Incomplete Filings)**
**Title:** New User Redirected to Complete Your Form 2290 Filing Page Upon Starting Filing

**Test Steps:**

1. Log in as a new user to the IRS e-file system.
2. Click on the "Start Filing" CTA for Form 2290.
3. Verify that the user is redirected to the /complete-your-form2290-filing page.
4. Enter incomplete filing information and click "Continue" button.
5. Verify that the user is directed back to the select form type page with highlighted incomplete sections.
6. Click on the "Start Filing" CTA again.
7. Verify that the user is redirected to the /complete-your-form2290-filing page for completion.

**Expected Result:**

* New users who have not completed a filing yet are directed to the complete-your-form2290-filing page upon clicking the start filing CTA from the select form type page.
* Existing users continue in the classic flow without interruption.
